== Description ==
El plugin Correo Direct para WordPress nos permite darnos de alta como afiliados en la plataforma de email marketing de Antevenio Correo Direct y empezar a ganar dinero como afiliados mediante el coregistro de usuarios en nuestro blog WordPress.


== Installation ==

Para instalar el plugin sigue estos pasos:

1. Descomprime el contenido del ZIP en la carpeta /plugins de tu blog
2. Activa el plugin en la sección “Plugins” del panel de administración de tu blog.
3. Nos damos de alta como afiliados en Correo Direct rellenando el fomulario que se encuentra en la sección    Ajustes>Correo Direct de nuestro blog.
4. Una vez dados de alta configuramos el plugin a nuestro gusto y especificamos nuestro identificador Feedburner que    explicamos más adelante.
5. Creamos una página de suscripción para nuestros usuarios.

El Plugin muestra a los usuarios un formulario de suscripción a la newsletter o boletín del blog junto al formulario de coregistro en Correo Direct.
La newsletter del blog es gestionada por Feedburner, así que tenéis que tener un identificador del blog para Feedburner y activada la opción de “mail subscription”. Los que ya lo uséis tendréis una dirección para el feed RSS de este estilo http://feeds.feedburner.com/TUIDENTIFICADOR

El Plugin cuenta con unas cuantas opciones de configuración como el identificador de feedburner y algunos títulos y textos personalizables.

Una vez configurado creamos una página nueva con un título del tipo “Suscripción al blog” o similar y en el contenido de la página simplemente incluiremos [correodirect]. Ya está, listo para funcionar. La nueva página mostrará el formulario mixto de suscripción a vuestro blog y coregistro en Correo Direct.

Como afiliados de Correo Direct contaréis con datos de acceso privados que recibiréis por correo electrónico tras rellenar el formulario de alta. Con estos datos podréis acceder a vuestro panel de afiliados donde veréis vuestras estadísticas e ingresos obtenidos.
